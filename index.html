How to Use

1. Build a team roster

  1. Select a team, or choose + Add Team.
  2. Choose Edit Team to rename it and add regular or substitute players.
  3. For each player, open Choose playable positions and check every position they can play. Rosters stay alphabetized and save automatically in this browser.
  4. Choose Done Editing, then use + Lineup beside the players and substitutes available for this game.

2. Set up the game

  • Select the game date and number of innings. Use + Add Game for a doubleheader; the new game keeps the previous kicking order but starts with blank fielding assignments.
  • Reorder players with the arrow buttons. The kicking tracker alternates female and male players while rotating each gender independently, so unequal roster sizes are supported.
  • Assign each player's position inning by inning, or choose Autogenerate Defense. The generator respects playable positions, keeps assignments you already entered, limits the field to five male players, and balances Bench time while preferring substitutes for extra Bench innings.

3. Lock and run the lineup

  • Every player needs a position or Bench in every inning before the lineup can be locked.
  • Once locked, use Kicked — Next to advance the current, on-deck, and in-the-hole kickers. Change Current inning as the game progresses; completed innings are hidden.
  • Unlock the lineup whenever you need to make changes. You can also clear only the fielding positions without changing the kicking order.

4. Save, print, back up, and share

  • Save stores all teams and lineups in this browser; Load restores that saved copy. Export a JSON backup to move everything to another device, and import it later.
  • Print / PDF creates a landscape page containing the lineup and fielding assignments.
  • Share Lineup creates a read-only live link. Share with Co-Captain creates a two-way editing link for the roster, lineup, inning, and kicking progress. Keep the owner's sharing page open while others are connected.
